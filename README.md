# Spring Boot HTTP Signature library

This is an implementation of https://tools.ietf.org/html/draft-cavage-http-signatures-12

Signatures supported:

- SHA256withRSA
- SHA256withRSA/PSS

Key stores supported:

- [ ] JKS
- [ ] PKSC 12

Headers supported:

- [ ] Signature header
- [x] Authorization header
